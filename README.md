# SQLite Analytics Challenge

## Overview
This project analyzes a relational database using SQLite. The goal is to answer business questions using SQL queries involving joins, grouping, filtering, and aggregation.

## Tools Used
- SQLite
- VS Code
- SQLTools extension


## Files Included
- challenge.sql → Contains all SQL queries for Tasks 1–4
- INSIGHTS.md → Business insights based on query results


## Tasks Completed

### Task 1: Top Customers by Spend
Identified the top 5 customers based on total spending.

### Task 2: Revenue by Product Category
Calculated total revenue generated by each product category.

### Task 3: Employee Salary Analysis
Compared employee salaries to department averages and identified above-average earners.

### Task 4: Loyalty by City
Analyzed distribution of Gold loyalty customers across cities.
